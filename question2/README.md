# Automated testing of MyObservatory's 9-day weather forecast

## Environment Setup
### 1. Basic dependency installation
- Install Python 3.8 or higher
- Install Poetry: pip install poetry
- Install Appium Server: npm install -g appium
- Start Appium Server: appium
- Launch an Android emulator or connect a physical device (with USB debugging enabled)
